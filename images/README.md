# Custom T-Shirt Images

To add your own t-shirt images to the store:

1. Place your image files in this directory (e.g., tshirt1.jpg, tshirt2.png)
2. Make sure your images are properly sized (recommended: 800x800 pixels)
3. Update the products array in script.js with your product information

Example:
```javascript
{
    id: 7,
    name: "Your Custom T-Shirt",
    price: 25.99,
    image: "images/your-tshirt.jpg"
}
```

Supported image formats: JPG, PNG, GIF, WEBP
